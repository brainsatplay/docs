    Libraries

    The Brains@Play Framework is dependent on several interrelated libraries.

    Library Status Description
    escompose Transforms ES Modules into linked Web Components.
    wasl The Web Application Specification Language.
    brainsatplay The Module Management System (MMS) for escompose and wasl applications.
    graphscript Simple graph-based programming in the browser and Node.js.
    visualscript Web Components for rapid prototyping
    datastreams-api Uniformly acquire real-time data with available browser APIs.
    tinybuild Custom build tool for web applications.
    escomposer A low-code editor for brainsatplay applications.
    @brainsatplay/components The official component registry for brainsatplay applications.
    @brainsatplay/docs Our documentation generator.
